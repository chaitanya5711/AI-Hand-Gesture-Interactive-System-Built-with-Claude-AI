# 🚀 AI Hand Gesture Interactive System

An advanced **real-time hand gesture interactive web application** that allows users to control visual effects using just their hands via webcam.

> ⚡ This project was fully generated with the help of Claude AI to demonstrate the real power of AI in software development.

---

## 🧠 About the Project

This project uses **computer vision and gesture recognition** to create an immersive interactive experience directly in the browser.

Users can perform hand gestures like pinch, fist, clap, etc., and control dynamic visual effects in real time.

---

## 🔥 Features

- ✋ Real-time hand tracking using webcam  
- 🎯 Gesture recognition (Pinch, Fist, Peace, Point, etc.)  
- 🎨 11+ interactive modes:
  - Elastic Bands  
  - Particle Flow  
  - Vortex  
  - Gravity Well  
  - Paint Mode  
  - Ripple Effects  
  - Constellation Mode  
- ⚡ Smooth animations and physics-based interactions  
- 🎥 Live video integration  
- 📸 Snapshot capture feature  
- 🎮 Gesture-based controls:
  - Pinch → Switch modes  
  - Clap → Explosion effect  
  - Double Fist → Slow motion
